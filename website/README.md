# Legacy static marketing site

> **Note:** The primary marketing experience now lives in the Next.js app at `http://localhost:3000` (`apps/web/src/app/(marketing)/`). This folder is kept for reference only.

## Local preview (optional)

```bash
npx serve website -p 4321
```

Then open [http://localhost:4321](http://localhost:4321).

## Recommended path

Run `make dev` and visit [http://localhost:3000](http://localhost:3000) for the full SaaS website with animations, product mockups, docs, FAQ, security, and pricing — plus CTAs to the prototype and app workspace.
